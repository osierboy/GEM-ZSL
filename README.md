**Title:**      "Goal-Oriented Gaze Estimation for Zero-Shot Learning"

**URL:**   [paper](https://openaccess.thecvf.com/content/CVPR2021/papers/Liu_Goal-Oriented_Gaze_Estimation_for_Zero-Shot_Learning_CVPR_2021_paper.pdf)
